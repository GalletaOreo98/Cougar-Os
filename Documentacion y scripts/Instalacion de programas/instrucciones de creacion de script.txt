*********CREACION DEL SCRIPT**************

1)Creacion de archivo de script.sh en terminal de linux o se crea a mano
	$ touch script.sh

2)lo abrimos con un editor de texto

3)insertamos las instrucciones
	#!/bin/sh
	sudo apt install anki
	sudo snap install notepadqq
	sudo rm script.sh
	reboot			

3) Darle permiso de ejecución al script, abre un Terminal y en la carpeta donde se encuentra el script digita:
	$ sudo chmod +x script.sh
